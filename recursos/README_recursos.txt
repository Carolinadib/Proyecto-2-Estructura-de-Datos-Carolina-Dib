Carpeta "recursos" - archivos de ejemplo para el proyecto Mendeley

Descripción:
Esta carpeta contiene archivos de texto que puedes copiar y pegar tal cual en archivos .txt para usar con la opción "Agregar resumen..." de la aplicación.

Instrucciones rápidas:
1. Abre el archivo que quieras usar (por ejemplo, `resumen_ejemplo1.txt`).
2. Selecciona todo su contenido (Ctrl+A) y cópialo (Ctrl+C).
3. Crea un archivo de texto en tu equipo (por ejemplo, `mi_resumen1.txt`) y pega (Ctrl+V) el contenido.
4. En la aplicación, usa "Agregar resumen..." y selecciona el archivo que creaste.
